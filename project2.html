Practice Task 2 - MS Excel Functions

In this project, I applied various Microsoft Excel functions to efficiently analyze and manipulate datasets. My tasks included:

  • Using formulas such as SUM, AVERAGE, and COUNT to calculate totals, averages, and counts of data entries
  • Applying IF, AND, OR logical functions to categorize and validate data
  • Employing VLOOKUP and HLOOKUP to search and retrieve specific data from large tables
  • Utilizing text functions like CONCAT, LEFT, RIGHT, and MID to clean and format textual data
  • Creating dynamic summaries and reports using PIVOT TABLES and conditional formatting

Through this project, I gained hands-on experience automating calculations, cleaning data, and generating insights using Excel functions, improving both efficiency and accuracy in data management.
